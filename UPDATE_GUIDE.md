# Life Dashboard 更新指南

你以后主要会改 3 类内容。

## 1. 改网站文字

打开 `index.html`，用搜索找到你想改的文字，直接替换即可。

常改位置：

- 首页主标题：`自己是最重要的产品`
- 个人身份：`商业化产品经理 / 职场自媒体博主 / 音乐内容输出者`
- 核心价值观：搜索 `Core Values`
- 生活现状：搜索 `Current State`
- 梦想路线图：搜索 `Dream Roadmap`
- 里程碑：搜索 `Milestones`
- 收藏库：搜索 `Signal Library`

## 2. 改复盘问题

打开 `script.js`，修改最上面的这组句子：

```js
const reviewQuestions = [
  "这个月，我有没有做一件能让我更自由的事？",
  "最近的行动，是在靠近愿景，还是只是在消耗惯性？"
];
```

注意每句话外面保留英文双引号，句子之间保留英文逗号。

## 3. 更换头像

把新头像命名为 `profile.jpg`，放进 `assets` 文件夹，替换原文件。

## 推荐更新节奏

- 每周：在里程碑或现状里补一条真实行动
- 每月：更新一次路线图和复盘问题
- 每季度：重写一次首页文字，看看你对自己的定义有没有变化

## GitHub Pages 自动更新方式

网站发布后，你每次在 GitHub 上修改文件并点击保存，GitHub Pages 都会自动更新网站。通常 1-2 分钟生效。

第一版不需要复杂后台。等你更新频率变高后，可以再升级成 Notion 或 Markdown 自动生成版本。
